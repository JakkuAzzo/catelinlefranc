üì∑ Image Optimization Tips

Before Uploading:

  1. Resize images to the exact dimensions shown above
  2. Compress using tools like:
    • TinyPNG - Web-based, easy to use
    • Squoosh - Advanced options
    • ImageOptim - Mac app
  3. Save in JPEG format (JPG) for photographs
  4. Quality settings: 80-85% for web use

üéØ Naming Convention

  • about-placeholder.jpg - About section photo
  • placeholder-1.jpg through placeholder-8.jpg - Portfolio grid images

Important: Filenames are case-sensitive and must match exactly!
