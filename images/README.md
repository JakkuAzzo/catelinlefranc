# Image Assets

Please add your photography images to this directory.

## Required Images

### Main Pages
- `hero-bg.jpg` - Hero section background (1920x1080px recommended)
- `about-placeholder.jpg` - About section image (600x800px recommended)

### Portfolio Items
- `placeholder-1.jpg` through `placeholder-8.jpg` - Portfolio grid images (800x1000px recommended, 4:5 ratio)

## Image Guidelines

### Formats
- Use `.jpg` for photographs
- Use `.png` for images requiring transparency
- Consider using `.webp` for better compression (with jpg fallback)

### Optimization
- Compress images before uploading
- Recommended tools: TinyPNG, ImageOptim, or Squoosh
- Target file size: Under 500KB for large images, under 200KB for portfolio items

### Dimensions
- **Hero Background**: 1920x1080px (16:9 ratio)
- **Portfolio Images**: 800x1000px (4:5 ratio)
- **About Image**: 600x800px (3:4 ratio)
- All images should be at least these dimensions for retina displays

### Naming Convention
- Use lowercase letters
- Use hyphens instead of spaces
- Be descriptive: `vintage-camera-1927.jpg` instead of `img1.jpg`
